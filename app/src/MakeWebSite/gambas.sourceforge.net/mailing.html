Mailing lists & Forums

Sourceforge Mailing-lists

Click on the link in the first column to go the mailman mailing-list interface where you will be able to subscribe to the list.

You can unsubscribe from the list in the last line of the mailing-list interface.

Mailing-list Description
Gambas Users This mailing-list is intended for gambas users, i.e. for people who write, or try to write programs with Gambas.
Gambas Developers This mailing-list is intended for gambas developers, i.e. for people who want to help me programming Gambas by writing components, and want to talk about that.
Gambas Subversion Commits This mailing-list is intended for gambas developers who want to be warned of each commit to the Gambas subversion repository.
Spanish Gambas Users This mailing-list is intended for spanish gambas users only.
French Gambas Users

This mailing-list is intended for french gambas users only.

If you want a mailing-list for your language, just ask me, I will add it.

All these mailing lists are located on Sourceforge servers, not on my own computer! So, if you have problems with posting to or receiving from a mailing-list, tell me, but understand that I will not necessarily help you if your problem is too basic.
